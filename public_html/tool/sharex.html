ShareX設定方法

ShareXの入手

https://getsharex.com/

設定方法

    1. 下記のコード、もしくは設定書き出しツールで生成したコードをコピー

    {
  "Name": "かんたんろだ",
  "RequestType": "POST",
  "RequestURL": "http://kntn.org/api/upload?jump=image",
  "FileFormName": "uploadfile",
  "Arguments": {},
  "ResponseType": "RedirectionURL",
  "RegexList": [],
  "URL": "",
  "ThumbnailURL": "",
  "DeletionURL": ""
}

    2. ShareXを起動

    3. メニューから [Destinations] -> [Custom uploaders] を選択

    4. [Import] -> [From Clipboard] を選択

    5. 設定画面を終了させる

    6. [Destinations] -> [Custom image uploader] を選択
